Modern AI = Prediction Machine

Not smart. Just really good at guessing:

  • What shape is this image?
  • What word comes next?
  • What image matches this caption?

Demo 1: Shape Recognition

  • Audience helps label: circle, triangle, square, star
  • Model learns to predict from patterns

Key Concepts

  • Input = image prompt (fixed size)
  • Output = probabilities over 4 classes
  • Training vs Inference
  • Model is fixed after training
  • Not a database — a mathematical formula

Demo 2: LLM Token Prediction

  • Input = sentence prompt
  • Output = probability distribution
  • Autoregressive: Output becomes new input

LLM Nuances

  • Token = subword (not always a full word)
  • Randomness settings: temperature, top-k, top-p
  • Hidden templates + role definitions
  • 200B+ parameters
  • Emergent behaviors
  • Takeaway: Don’t blindly trust outputs

Demo 3: Simple Image Generation

  • Reverse the classifier
  • Generate images the model classifies confidently
  • Generation is guided by prediction

Demo 4: Image Generation at Scale

  • Stable Diffusion or Flux Model
  • Trained on text + image pairs
  • Every image is unique
  • 14B parameters, needs 56GB VRAM

Wrap-up: Key Takeaways

  • AI = pattern recognizer
  • Training = learn patterns
  • Inference = apply them
  • Prompting = guide the pattern generation

Using AI in Teaching

  • Prompt-based feedback (OpenAI Playground)
  • Lesson planning aid
  • Language, visuals, and coding support

Cautions & Opportunities

  • AI helps, but doesn’t replace expertise
  • Outputs are not always reliable
  • Your knowledge improves your prompting
